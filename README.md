# Coordinate Measuring Machine (CMM) Sensor Builder

This is a console-based C++ application that allows users to select components for a Coordinate Measuring Machine (CMM) and calculates the total length and CTE (Coefficient of Thermal Expansion).

## Features
- Select CMM components interactively
- Calculates total length
- Calculates Coefficient of Thermal Expansion
- Console-based, simple interface

## Technologies
- C++
- Console application

## How to Run
1. Open the project in a C++ IDE or compile using g++
2. Run the executable and follow the prompts
