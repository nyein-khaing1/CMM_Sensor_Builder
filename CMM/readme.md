CMM Sensor Builder

Developer - Nyein Ei Khaing
Time Spent - Approximately 2 hrs

Overview
This is a console-based C++ application that allows users to construct Coordinate Measuring Machine (CMM) sensors by selecting components from a predefined catalogue.
The application calculates the overall length and Coefficient of Thermal Expansion (CTE) of the constructed sensor.


Features
- Component Catalogue: View all available components (Head, Probe, Module, Tip)
- Sensor Construction: Select components to build a complete sensor
Automatic Calculations:
- Total length (sum of all component lengths)
- Overall CTE (weighted average based on component lengths)
- Configuration Management: View current sensor setup and clear selections
- User-Friendly Interface: Menu-driven console application


Testing
